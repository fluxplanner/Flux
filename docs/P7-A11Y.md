# P7-A11Y

**Step ID:** `P7-A11Y`  
**Flag:** `enable_a11y_suite` (default **off**)

Unified accessibility preferences: reuses existing reduce motion, font scale, and high contrast, and adds **personal calm mode** and **ADHD-friendly focus layout** (client-side only).

## Preferences (localStorage)

| Key | Effect |
|-----|--------|
| `flux_reduce_motion` | `html.flux-reduce-motion` (existing) |
| `flux_font_scale` | Root font scale % (existing) |
| `flux_high_contrast` | CSS variables + `body.high-contrast` + `html.flux-high-contrast` |
| `flux_personal_calm_mode` | Softer colors, hides mesh/cursor effects |
| `flux_adhd_focus_mode` | Simpler dashboard, larger tap targets, stronger focus rings |

Personal calm is separate from school **emergency calm** (`flux-school-emergency.js`).

## Modules

| File | Role |
|------|------|
| `public/js/flux-a11y.js` | `FluxA11y` — apply prefs, settings UI |
| `public/css/flux-a11y.css` | Calm / ADHD / high-contrast helpers |
| `supabase/migrations/20260525450000_a11y_suite.sql` | Feature flag seed |

## Enable (dev)

```javascript
window.FLUX_EXPERIMENTS = { enable_a11y_suite: true };
await FluxFeatureFlags.load({ force: true });
FluxA11y.install();
```

Settings → **Look** → Accessibility suite card (calm + ADHD toggles). Reduce motion and high contrast remain in the same pane.

## Rollback

Disable flag; calm/ADHD classes are not applied; localStorage keys remain harmless.
