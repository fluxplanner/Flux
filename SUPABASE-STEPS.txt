  FLUX PLANNER — SUPABASE (DO THIS ONCE, IN ORDER)
  You do not need to understand the text. Just click and paste.
═══════════════════════════════════════════════════════════════

WHAT YOU NEED
  • A Supabase account (the website where your app database lives)
  • The file named:  PASTE-INTO-SUPABASE.sql
    It is in the same folder as this file (your Flux Planner folder).

───────────────────────────────────────────────────────────────
STEP 1 — OPEN SUPABASE IN YOUR BROWSER
───────────────────────────────────────────────────────────────
  1. Go to:  https://supabase.com
  2. Sign in (same way you usually sign in).

───────────────────────────────────────────────────────────────
STEP 2 — OPEN YOUR PROJECT
───────────────────────────────────────────────────────────────
  3. Click the project you use for Flux Planner (the one you already set up).
     If you only have one project, click that one.

───────────────────────────────────────────────────────────────
STEP 3 — OPEN THE SQL EDITOR
───────────────────────────────────────────────────────────────
  4. On the left side, look for an icon or word:  SQL
  5. Click it.
  6. Click:  New query   (or a blank query tab).

───────────────────────────────────────────────────────────────
STEP 4 — COPY THE BIG SQL FILE
───────────────────────────────────────────────────────────────
  7. On your Mac, open Finder.
  8. Go to:  Desktop  →  Flux Planner   (or wherever this folder is).
  9. Double-click the file:  PASTE-INTO-SUPABASE.sql
     It may open in TextEdit, Notes, or Cursor — any is fine.
 10. Select EVERYTHING in that file:
       • Click inside the file, then press:  Command + A
 11. Copy it:
       • Press:  Command + C

───────────────────────────────────────────────────────────────
STEP 5 — PASTE AND RUN IN SUPABASE
───────────────────────────────────────────────────────────────
 12. Click inside the empty SQL box in Supabase (the website).
 13. Paste:
       • Press:  Command + V
 14. Find the green button that says  Run   (or  RUN  /  Execute).
 15. Click  Run  once.
 16. Wait until it finishes (a few seconds).

───────────────────────────────────────────────────────────────
STEP 6 — CHECK THAT IT WORKED
───────────────────────────────────────────────────────────────
 17. Under the SQL box, you should see a success message.
     • Words like “Success” or “No rows returned” are OK.
 18. If you see RED text that says ERROR:
       • Do not click Run again yet.
       • Take a screenshot and send it to whoever helps you with Flux,
         or copy the red error text.

───────────────────────────────────────────────────────────────
IMPORTANT
───────────────────────────────────────────────────────────────
  • Running this whole file more than once is OK for this project
    (it is written to be safe to run again).
  • You only need to do STEPS 1–16 once (unless you get an error).
